# Java Fundamentals

This module covers the core building blocks of the Java programming language. It introduces essential concepts such as data types, variables, operators, control flow, and type conversions that form the foundation for writing Java programs.

The topics in this module provide the knowledge required before moving on to Object-Oriented Programming and other advanced Java concepts.

---

## 📚 Topics Covered

### 1. Data Types

Learn about the different types of data supported by Java.

Topics include:

- Primitive Data Types
- Non-Primitive Data Types
- Variables
- Literals
- Primitive vs Reference Types

📄 **Folder:** `01-Data-Types`

---

### 2. Type Conversion & Type Casting

Understand how Java converts one data type to another.

Topics include:

- Implicit Type Conversion (Widening)
- Explicit Type Casting (Narrowing)
- Data Loss
- Casting Examples

📄 **Folder:** `02-Type-Conversion-and-Casting`

---

### 3. Type Promotion

Learn how Java automatically promotes data types during expressions.

Topics include:

- Unary Promotion
- Binary Promotion
- Arithmetic Expressions
- Type Promotion Rules

📄 **Folder:** `03-Type-Promotion`

---

### 4. Operators

Learn the different operators available in Java.

Topics include:

- Arithmetic Operators
- Unary Operators
- Assignment Operators
- Relational Operators
- Logical Operators
- Bitwise Operators
- Shift Operators
- Ternary Operator
- Operator Precedence

📄 **Folder:** `04-Operators`

---

### 5. Variables in Java

Learn how variables are declared, stored, and used in Java programs.

Topics include:

- Types of Variables
- Local Variables
- Instance Variables
- Class (Static) Variables
- Memory Allocation
- Reference Variables
- Variable Scope
- Default Values

📄 **Folder:** `05-Variables-in-Java`

---

### 6. Conditional Statements

Learn how to control program execution using decision-making statements.

Topics include:

- `if`
- `if-else`
- Nested `if`
- `if-else-if`
- `switch`
- Enhanced `switch`

📄 **Folder:** `06-Conditional-Statements`

---

### 7. Loops

Learn how to execute a block of code repeatedly using loops.

Topics include:

- `for` Loop
- `while` Loop
- `do-while` Loop
- Enhanced `for` Loop
- Nested Loops
- `break`
- `continue`

📄 **Folder:** `07-Loops`

---

## 🎯 Learning Outcome

After completing this module, you will be able to:

- Understand Java data types and variables.
- Perform type conversion and type casting.
- Explain Java's type promotion rules.
- Use different operators effectively.
- Understand variable scope, memory allocation, and reference variables.
- Implement decision-making using conditional statements.
- Write programs using loops and control flow statements.
