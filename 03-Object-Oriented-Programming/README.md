# Object-Oriented Programming (OOP)

This module introduces the core Object-Oriented Programming (OOP) concepts in Java. OOP helps organize programs using classes and objects, making code more modular, reusable, maintainable, and scalable.

The topics in this module provide the foundation for designing real-world applications using object-oriented principles.

---

## 📚 Topics Covered

### 1. Class and Object

Learn the basic building blocks of Object-Oriented Programming.

Topics include:

- What is a Class?
- What is an Object?
- Creating Classes
- Creating Objects
- Accessing Members
- Object Creation Process

📄 **Folder:** `01-Class-and-Object`

---

### 2. Method Overloading

Understand how Java supports compile-time polymorphism through method overloading.

Topics include:

- What is Method Overloading?
- Rules for Method Overloading
- Different Number of Parameters
- Different Data Types
- Different Parameter Order
- Method Resolution

📄 **Folder:** `02-Method-Overloading`

---

## 🎯 Learning Outcome

After completing this module, you will be able to:

- Understand the fundamentals of Object-Oriented Programming
- Create classes and objects
- Access object members and methods
- Understand object creation in Java
- Implement method overloading
- Differentiate overloaded methods based on their parameters
