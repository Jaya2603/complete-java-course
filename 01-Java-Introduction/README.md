# Java Introduction

This module introduces the fundamental concepts required to start programming in Java. It covers setting up the Java development environment, understanding how Java programs are compiled and executed, exploring the Java runtime architecture, and writing the first Java program.

The topics in this module provide the foundation for learning Core Java and understanding how Java applications work internally.

---

## 📚 Topics Covered

### 1. Java Installation & Setup

Learn how to set up the Java development environment.

Topics include:

- Installing JDK
- Choosing an IDE
- Setting up `JAVA_HOME`
- Configuring environment variables
- Verifying Java installation

📄 **File:** `01-Java-Installation-Setup.md`

---

### 2. JShell

Introduction to Java's Read-Eval-Print Loop (REPL).

Topics include:

- What is JShell
- Running Java code interactively
- Testing code snippets
- Useful JShell commands

📄 **File:** `02-JShell.md`

---

### 3. Java Program Execution Flow

Understand how a Java program executes from source code to output.

Topics include:

- Java source code
- Java compiler (`javac`)
- Bytecode generation
- JVM execution
- Operating System interaction
- Hardware execution
- Platform independence

📄 **File:** `03-Java-Program-Execution-Flow.md`

---

### 4. Hello World Program – Internal Working

Learn what happens internally when a simple Java program runs.

Topics include:

- Compilation
- Class loading
- Bytecode verification
- JVM memory allocation
- Execution Engine
- `System.out.println()`

Includes execution flow diagrams.

📄 **File:** `04-Hello-World-Program-Internal-Working.md`

---

### 5. JDK, JRE and JVM

Understand the Java platform architecture and runtime environment.

Topics include:

- Java Development Kit (JDK)
- Java Runtime Environment (JRE)
- Java Virtual Machine (JVM)
- Components of JRE
- Java Class Libraries
- Runtime architecture
- Differences between JDK, JRE and JVM

Includes architecture diagrams.

📄 **File:** `05-JDK-JRE-JVM.md`

---

## 🎯 Learning Outcome

After completing this module, you will be able to:

- Set up a Java development environment
- Write and execute Java programs
- Understand the Java compilation and execution process
- Explain the roles of JDK, JRE, and JVM
- Understand the internal workflow of a Java application
